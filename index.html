Projects

Solar Lander

About: Solar Lander is a libGDX game-engine based n-body/orbit simulation that features a small controllable lander the user can pilot around and use to interact with the gravitational forces of the randomly generated celestial bodies

Language: Java

Software: IntelliJ Idea, Aseprite

Mouse Maze

About: Mouse Maze is a MonoGame engine project that uses pathfinding and maze generation where the user can spawn in both mice and cheese then watch them navigate themselves around the labyrinth in search of Swiss cheese

Language: C#

Software: Visual Studio 2022, Aseprite

Particle Sim

About: Particle Sim is a libGDX game-engine particle simulation project that attempts to recreate falling sand physics in a grid-based system. The user can spawn in particles by left clicking on the grid

Language: Java

Software: IntelliJ Idea

Particle Sim 2

About: Particle Sim 2 is a MonoGame based remake of my libGDX falling sand particle simulation. Like the first particle sim, the user can spawn in new particles by left clicking to interact with the world

Language: C#

Software: Visual Studio 2022

BlackJack

About: BlackJack is a minigame recreation of the regular black jack card game in the libGDX engine, the user can interact with two buttons to choose whether to hit or stay

Language: Java

Software: IntelliJ Idea

Qud Mycologist

About: Qud Mycologist is a small mod for the science fantasy roguelike game, Caves of Qud. The mod adds a new profession (also called subtype) for the player to choose from. In this case, the profession is a mycologist and one who is too familiar with fungi. The player has the unique start of having multiple different fungi infections

Languages: C#, XML, JSON

Software: Visual Studio Code
